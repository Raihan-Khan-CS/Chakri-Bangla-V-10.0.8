সন্ধি: সংজ্ঞা ও প্রকারভেদ

সন্ধি শব্দের অর্থ মিলন। ব্যাকরণে, সন্নিহিত (পাশাপাশি অবস্থিত) দুটি ধ্বনির মিলনকে সন্ধি বলে। মূলত, উচ্চারণের সুবিধার জন্য এবং ধ্বনিগত মাধুর্য সৃষ্টির জন্য সন্ধি করা হয়। যেমন: 'বিদ্যা' এবং 'আলয়' এই দুটি শব্দ উচ্চারণে যে সময় লাগে, 'বিদ্যালয়' উচ্চারণে তার চেয়ে কম সময় লাগে এবং শুনতে শ্রুতিমধুর হয়।

বিসিএস পরীক্ষার জন্য সন্ধি একটি অপরিহার্য অধ্যায়। সন্ধি বিচ্ছেদ এবং সন্ধি গঠন উভয় প্রকার প্রশ্নই এসে থাকে।

সন্ধির প্রকারভেদ

বাংলা ভাষার সন্ধি প্রধানত দুই প্রকার:

  • তৎসম বা সংস্কৃত সন্ধি: সংস্কৃত ভাষা থেকে আগত এবং সংস্কৃত নিয়ম মেনে গঠিত সন্ধি। এটি তিন প্রকার:
    • স্বরসন্ধি: স্বরধ্বনির সাথে স্বরধ্বনির মিলন।
    • ব্যঞ্জনসন্ধি: স্বরধ্বনির সাথে ব্যঞ্জনধ্বনি, ব্যঞ্জনধ্বনির সাথে স্বরধ্বনি অথবা ব্যঞ্জনধ্বনির সাথে ব্যঞ্জনধ্বনির মিলন।
    • বিসর্গ সন্ধি: বিসর্গের (ঃ) সাথে স্বরধ্বনি বা ব্যঞ্জনধ্বনির মিলন।
  • খাঁটি বাংলা সন্ধি: তদ্ভব ও দেশি শব্দের সন্ধি, যা মূলত উচ্চারণের সুবিধার ওপর নির্ভরশীল। এটি দুই প্রকার:
    • বাংলা স্বরসন্ধি
    • বাংলা ব্যঞ্জনসন্ধি

তৎসম স্বরসন্ধি

স্বরধ্বনির সঙ্গে স্বরধ্বনির মিলনে যে সন্ধি হয়, তাকে স্বরসন্ধি বলে।

সূত্র: অ-কার বা আ-কারের পর ই-কার বা ঈ-কার থাকলে উভয়ে মিলে এ-কার (ে) হয়।

উদাহরণ:
(অ + ই = এ): শুভ + ইচ্ছা = শুভেচ্ছা, নর + ইন্দ্র = নরেন্দ্র, দেব + ইন্দ্র = দেবেন্দ্র, পূর্ণ + ইন্দু = পূর্ণেন্দু, স্বেচ্ছা (স্ব + ইচ্ছা), गज + इंद्र = गजेंद्र, ইত্যাদি।
(অ + ঈ = এ): নর + ঈশ = নরেশ, পরম + ঈশ্বর = পরমেশ্বর, গণ + ঈশ = গণেশ, দিন + ঈশ = দীনেশ, ইত্যাদি।
(আ + ই = এ): যথা + ইষ্ট = যথেষ্ট, মহা + ইন্দ্র = মহেন্দ্র, তথা + ইতি = তথেতি, ইত্যাদি।
(আ + ঈ = এ): মহা + ঈশ = মহেশ, রমা + ঈশ = রমেশ, ঢাকা + ঈশ্বরী = ঢাকেশ্বরী, ইত্যাদি।

৫. অ/আ + উ/ঊ = ও (ো)

সূত্র: অ-কার বা আ-কারের পর উ-কার বা ঊ-কার থাকলে উভয়ে মিলে ও-কার (ো) হয়।

উদাহরণ:
(অ + উ = ও): সূর্য + উদয় = সূর্যোদয়, পর + উপকার = পরোপকার, চল + ঊর্মি = চলোর্মি, ফল + উদয় = ফলোদয়, হিত + উপদেশ = হিতোপদেশ, প্রশ্ন + উত্তর = প্রশ্নোত্তর, ইত্যাদি।
(অ + ঊ = ও): গৃহ + ঊর্ধ্ব = গৃহোর্ধ্ব, নব + ঊঢ়া = নবোঢ়া, ইত্যাদি।
(আ + উ = ও): মহা + উৎসব = মহোৎসব, যথা + উচিত = যথোচিত, গঙ্গা + উদক = গঙ্গোদক, ইত্যাদি।
(আ + ঊ = ও): মহা + ঊর্মি = মহোর্মি, গঙ্গা + ঊর্মি = গঙ্গোর্মি, ইত্যাদি।

৬. অ/আ + ঋ = অর

সূত্র: অ-কার বা আ-কারের পর ঋ-কার থাকলে উভয়ে মিলে 'অর' হয় এবং তা রেফ (র্) রূপে পরবর্তী বর্ণের সাথে যুক্ত হয়।

উদাহরণ:
(অ + ঋ = অর): দেব + ঋষি = দেবর্ষি, সপ্ত + ঋষি = সপ্তর্ষি, অধম + ঋণ = অধমর্ণ, উত্তম + ঋণ = উত্তমর্ণ, ইত্যাদি।
(আ + ঋ = অর): মহা + ঋষি = মহর্ষি, রাজা + ঋষি = রাজর্ষি, ক্ষুধা + ঋত = ক্ষুধার্ত, তৃষ্ণা + ঋত = তৃষ্ণার্ত, ইত্যাদি।

৭. অ/আ + এ/ঐ = ঐ (ৈ)

সূত্র: অ-কার বা আ-কারের পর এ-কার বা ঐ-কার থাকলে উভয়ে মিলে ঐ-কার (ৈ) হয়।

উদাহরণ:
(অ + এ = ঐ): জন + এক = জনৈক, হিত + এষী = হিতৈষী, মত + ঐক্য = মতৈক্য, সর্ব + এব = সর্বৈব, ইত্যাদি।
(আ + এ = ঐ): সদা + এব = সদৈব, তথা + এব = তথৈব, ইত্যাদি।
(অ + ঐ = ঐ): মত + ঐক্য = মতৈক্য, দেব + ঐশ্বর্য = দেবৈশ্বর্য, ইত্যাদি।
(আ + ঐ = ঐ): মহা + ঐশ্বর্য = মহৈশ্বর্য, মহা + ঐরাবত = মহৈরাবত, ইত্যাদি।

৮. অ/আ + ও/ঔ = ঔ (ৌ)

সূত্র: অ-কার বা আ-কারের পর ও-কার বা ঔ-কার থাকলে উভয়ে মিলে ঔ-কার (ৌ) হয়।

উদাহরণ:
(অ + ও = ঔ): বন + ওষধি = বনৌষধি, জল + ওকা = জলৌকা, ইত্যাদি।
(আ + ও = ঔ): মহা + ওষধি = মহৌষধি, ইত্যাদি।
(অ + ঔ = ঔ): পরম + ঔষধ = পরমৌষধ, চিত্ত + ঔদার্য = চিত্তৌদার্য, ইত্যাদি।
(আ + ঔ = ঔ): মহা + ঔষধ = মহৌষধ, মহা + ঔদার্য = মহৌদার্য, ইত্যাদি।

৯. ই/ঈ + ভিন্ন স্বর = য-ফলা (্য)

সূত্র: ই-কার বা ঈ-কারের পর অন্য কোনো স্বরধ্বনি থাকলে ই/ঈ স্থানে য-ফলা (্য) হয়।

উদাহরণ:
অতি + অন্ত = অত্যন্ত, যদি + অপি = যদ্যপি, ইতি + আদি = ইত্যাদি, প্রতি + এক = প্রত্যেক, অতি + আচার = অত্যাচার, অধি + উষ্য = অধ্যুষ্য, প্রতি + ঊষ = प्रत्यूष, নদী + অম্বু = নদ্যম্বু, মসী + আধার = মস্যাধার, ইত্যাদি।

১০. উ/ঊ + ভিন্ন স্বর = ব-ফলা (্ব)

সূত্র: উ-কার বা ঊ-কারের পর অন্য কোনো স্বরধ্বনি থাকলে উ/ঊ স্থানে ব-ফলা (্ব) হয়।

উদাহরণ:
সু + অল্প = স্বল্প, অনু + ইত = অন্বিত, তনু + ঈ = তন্বী, মনু + অন্তর = মন্বন্তর, সু + আগত = স্বাগত, অনু + এষণ = অন্বেষণ, ইত্যাদি।

১১. ঋ + ভিন্ন স্বর = র-ফলা (্র)

সূত্র: ঋ-কারের পর অন্য কোনো স্বরধ্বনি থাকলে ঋ স্থানে র-ফলা (্র) হয়।

উদাহরণ:
পিতৃ + আলয় = পিত্রালয়, মাতৃ + আদেশ = মাত্রাদেশ, পিতৃ + ইচ্ছা = পিত্রচ্ছা, মাতৃ + উপদেশ = মাত্রুপদেশ, ইত্যাদি।

১২. এ/ঐ/ও/ঔ + স্বরধ্বনি

সূত্র: এ, ঐ, ও, ঔ-কারের পর কোনো স্বরধ্বনি থাকলে এরা যথাক্রমে অয়, আয়, অব, আব হয়।

উদাহরণ:
(এ > অয়): নে + অন = নয়ন, শে + অন = শয়ন, চে + অন = চয়ন।
(ঐ > আয়): নৈ + অক = নায়ক, গৈ + অক = গায়ক, সৈ + অক = সায়ক।
(ও > অব): পো + অন = পবন, লো + অন = লবণ, ভো + অন = ভবন।
(ঔ > আব): পৌ + অক = পাবক, নৌ + ইক = নাবিক, ভৌ + উক = ভাবুক।

তৎসম ব্যঞ্জনসন্ধি

স্বরধ্বনির সাথে ব্যঞ্জনধ্বনি, ব্যঞ্জনধ্বনির সাথে স্বরধ্বনি কিংবা ব্যঞ্জনধ্বনির সাথে ব্যঞ্জনধ্বনির মিলনে যে সন্ধি হয়, তাকে ব্যঞ্জনসন্ধি বলে।

১. স্বরধ্বনি + ব্যঞ্জনধ্বনি

সূত্র: স্বরধ্বনির পর ছ থাকলে, ছ স্থানে চ্ছ হয়।

উদাহরণ:
এক + ছত্র = একচ্ছত্র, আ + ছাদন = আচ্ছাদন, পরি + ছেদ = পরিচ্ছেদ, বি + ছেদ = বিচ্ছেদ, তরু + ছায়া = তরুচ্ছায়া, মুখ + ছবি = মুখচ্ছবি, কথা + ছলে = কথাচ্ছলে, ইত্যাদি।

২. ব্যঞ্জনধ্বনি + স্বরধ্বনি

সূত্র: বর্গের প্রথম ধ্বনির (ক্, চ্, ট্, ত্, প্) পর স্বরধ্বনি থাকলে, প্রথম ধ্বনিটি তার বর্গের তৃতীয় ধ্বনিতে (গ্, জ্, ড্, দ্, ব্) পরিণত হয়।

উদাহরণ:
দিক্ + অন্ত = দিগন্ত, বাক্ + ঈশ = বাগীশ, ষট্ + আনন = ষড়ানন, তৎ + অবধি = তদবধি, সুপ্ + অন্ত = সুবন্ত, দিক্ + বিজয় = দিগ্বিজয়, সৎ + উপায় = সদুপায়, জগৎ + ইন্দ্র = জগদিন্দ্র, সৎ + আনন্দ = সদানন্দ, ইত্যাদি।

৩. ব্যঞ্জনধ্বনি + ব্যঞ্জনধ্বনি

ক) ত্/দ্ + চ্/ছ্ = চ্ছ: সৎ + ছাত্র = সচ্ছাত্র, উৎ + ছেদ = উচ্ছেদ, শরৎ + চন্দ্র = শরচ্চন্দ্র, সৎ + চরিত্র = সচ্চরিত্র, বিপদ + চয় = বিপচ্চয়।

খ) ত্/দ্ + জ্/ঝ্ = জ্জ: সৎ + জন = সজ্জন, জগৎ + জীবন = জগজ্জীবন, বিপদ + জাল = বিপজ্জাল, উৎ + জল = উজ্জ্বল, কুৎ + ঝটিকা = কুজ্ঝটিকা।

গ) ত্/দ্ + ড্/ঢ্ = ড্ড: উৎ + ডীন = উড্ডীন, বৃহৎ + ঢক্কা = বৃহড্ঢক্কা।

ঘ) ত্/দ্ + ল্ = ল্ল: উৎ + লাস = উল্লাস, উৎ + লেখ = উল্লেখ, তৎ + লিপি = তল্লিপি।

ঙ) ত্/দ্ + শ্ = চ্ছ: উৎ + শ্বাস = উচ্ছ্বাস, চলৎ + শক্তি = চলচ্ছক্তি, শরৎ + শশী = শরচ্ছশী।

চ) ত্/দ্ + হ্ = দ্ধ: উৎ + হার = উদ্ধার, পদ্ + হতি = পদ্ধতি, উৎ + হৃত = উদ্ধৃত।

ছ) ম্ + স্পর্শবর্ণ = ম্ স্থানে সেই বর্গের পঞ্চম বর্ণ (ঙ, ঞ, ণ, ন, ম) হয়।
সম্ + গীত = সঙ্গীত, সম্ + চয় = সঞ্চয়, সম্ + তাপ = সন্তাপ, সম্ + পূর্ণ = সম্পূর্ণ, সম্ + মান = সম্মান, অহম্ + কার = অহংকার, কিম্ + নর = কিন্নর, সম্ + ধান = সন্ধান, সম্ + न्यासी = সন্ন্যাসী।

জ) বর্গের প্রথম বর্ণ + পঞ্চম বর্ণ = প্রথম বর্ণ স্থানে তৃতীয় বা পঞ্চম বর্ণ হয়।
বাক্ + ময় = বাঙ্ময়, দিক্ + নির্ণয় = দিঙ্নির্ণয়, জগৎ + নাথ = জগন্নাথ, উৎ + নয়ন = উন্নয়ন, মৃৎ + ময় = মৃন্ময়, চিৎ + ময় = চিন্ময়।

বিসর্গ সন্ধি

বিসর্গের (ঃ) সাথে স্বরধ্বনি বা ব্যঞ্জনধ্বনির মিলনে যে সন্ধি হয়, তাকে বিসর্গ সন্ধি বলে। বিসর্গ মূলত 'র্' এবং 'স্' এর সংক্ষিপ্ত রূপ।

১. বিসর্গের পর স্বরধ্বনি

সূত্র: অ-কারের পরস্থিত বিসর্গের পর অ ধ্বনি থাকলে, বিসর্গ ও অ মিলে ও-কার (ো) হয়।

উদাহরণ:
ততঃ + অধিক = ততোধিক, মনঃ + অভিলষিত = মনোভিলাষিত, বয়ঃ + অধিক = বয়োধিক, যশঃ + অভিলাষ = যশোভিলাষ।

২. বিসর্গের পর ব্যঞ্জনধ্বনি

ক) বিসর্গের পর চ/ছ থাকলে বিসর্গ স্থানে 'শ' হয়।
নিঃ + চয় = निश्चय, শিরঃ + ছেদ = শিরশ্ছেদ, দুঃ + চরিত্র = দুশ্চরিত্র।

খ) বিসর্গের পর ট/ঠ থাকলে বিসর্গ স্থানে 'ষ' হয়।
ধনুঃ + টঙ্কার = ধনুষ্টঙ্কার, নিঃ + ঠুর = নিষ্ঠুর।

গ) বিসর্গের পর ত/থ থাকলে বিসর্গ স্থানে 'স' হয়।
নমঃ + কার = নমস্কার, দুঃ + তর = দুস্তর, মনঃ + তাপ = মনস্তাপ, শিরঃ + ত্রাণ = শিরস্ত্রাণ।

ঘ) অ/আ ভিন্ন স্বরের পর বিসর্গ এবং পরে স্বরধ্বনি/বর্গের ৩য়/৪র্থ/৫ম ধ্বনি/য/র/ল/হ থাকলে বিসর্গ স্থানে 'র' হয়।
নিঃ + আকার = নিরাকার, দুঃ + যোগ = দুর্যোগ, আশীঃ + বাদ = আশীর্বাদ, দুঃ + অবস্থা = দুরবস্থা, অন্তঃ + গত = অন্তর্গত, পুনঃ + আয় = পুনরায়।

ঙ) অ-কারের পর বিসর্গ এবং পরে বর্গের ৩য়/৪র্থ/৫ম ধ্বনি/য/র/ল/হ থাকলে বিসর্গ স্থানে 'ও' হয়।
মনঃ + রম = মনোরম, সদ্যঃ + জাত = সদ্যোজাত, মনঃ + গত = মনোগত, তিরঃ + ধান = তিরোধান, মনঃ + যোগ = মনোযোগ।

নিপাতনে সিদ্ধ সন্ধি

যেসব সন্ধি কোনো নিয়ম মানে না, ব্যতিক্রমীভাবে গঠিত হয়, তাদের নিপাতনে সিদ্ধ সন্ধি বলে।

স্বরসন্ধি: কুল + অটা = কুলটা, গো + অক্ষ = গবাক্ষ, প্র + ঊঢ় = প্রৌঢ়, অন্য + অন্য = অন্যান্য, মার্ত + অণ্ড = মার্তণ্ড, শুদ্ধ + ওদন = শুদ্ধোদন।

ব্যঞ্জনসন্ধি: আ + চর্য = আশ্চর্য, গো + পদ = গোষ্পদ, এক + দশ = একাদশ, ষট্ + দশ = ষোড়শ, বৃহৎ + পতি = বৃহস্পতি, তৎ + কর = তস্কর, পর + পর = পরস্পর, মনস্ + ঈষা = মনীষা, পতৎ + অঞ্জলি = পতঞ্জলি।

বিসর্গ সন্ধি: অহঃ + অহঃ = অহরহ, অহঃ + নিশ = অহর্নিশ, ভাঃ + কর = ভাস্কর, বাচঃ + পতি = বাচস্পতি।

খাঁটি বাংলা সন্ধি

তদ্ভব, দেশি ও বিদেশি শব্দের সন্ধি মূলত উচ্চারণের সহজপ্রবণতার উপর নির্ভরশীল এবং কোনো নির্দিষ্ট ব্যাকরণগত নিয়ম কঠোরভাবে অনুসরণ করে না।

বাংলা স্বরসন্ধি

স্বরলোপ: কাঁচা + কলা = কাঁচকলা, নাতি + বউ = নাতবউ, ঘোড়া + গাড়ি = ঘোড়াগাড়ি, মিথ্যা + উক = মিথ্যুক, শতেক (শত + এক), রুপা + আলি = রুপালি, ইত্যাদি।

বাংলা ব্যঞ্জনসন্ধি

ব্যঞ্জন লোপ: পাঁচ + সের = পাঁসের, ছোট + দা = ছোড়দা, রাঁধ + না = রান্না, ইত্যাদি।
অন্যান্য: আর + না = আরনা, চার + টি = চারটি, কাঁচা + তে = কাঁচাতে, ইত্যাদি।
